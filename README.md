# DEVELOPMENT-OF-A-SOUND-MASTER-PLANNING-METHODOLOGY-CONSIDERING-LOCAL-FEATURES
Тема: Разработка методики проектирования звукового мастер-плана с учетом локальных особенностей

Автор: Федина Ксения

Описание работы: Звуковой мастер-план с учетом локальных особенностей территории Бульвара Белых Ночей, п. Новоселье

## Описание структуры репозитория

Репозиторий `sound-masterplan-local-features` содержит материалы выпускной квалификационной работы, посвященной разработке методики проектирования звукового мастер-плана с учетом локальных особенностей территории. В состав репозитория входят пространственные данные, проект QGIS, стили оформления слоев, результаты опроса жителей, итоговые карты, схемы, иллюстрации, фотоматериалы и текстовые материалы по теме исследования.

Структура репозитория организована по типам материалов: пространственные данные размещены в папке `data`, проектные файлы QGIS — в папке `qgis`, итоговые графические материалы — в папке `outputs`, текстовые документы — в папке `docs`, а материалы натурного обследования — в папке `media`.

    ## Описание исходных данных и метода

Исследование основано на сочетании пространственных, акустических, натурных и социологических данных, собранных и подготовленных для разработки звукового мастер-плана территории.

К исходным данным относятся границы территории, функциональное зонирование, локальные ограничения по шуму, источники шумового воздействия, точки замеров шума, материалы натурного обследования, данные звукового картирования, проектная документация и результаты опроса жителей.

Предложенный метод объединяет нормативную оценку шумовой нагрузки, звуковое картирование, анализ локальных особенностей территории, исследование пользовательского восприятия и сценарное проектирование звуковой среды. Методика позволяет рассматривать звук не только как источник шумового воздействия, но и как проектный параметр, связанный с комфортом, безопасностью и восстановительным потенциалом городской среды.

Метод был апробирован на территории Бульвара Белых Ночей в п. Новоселье. В рамках апробации рассматривались три типа территориальных условий: существующая территория, доступная для натурного обследования; реализованная территория с проектной документацией; территория в процессе строительства, анализируемая преимущественно по проектным материалам.
